Sistema de Aulas - Tema Futurista Translucido
-----------------------------------------------
Contiene:
- index.html (login)
- dashboard.html (mapa arquitectónico con 5 módulos × 6 aulas)
- aula.html (detalle, reservas, recursos, fotos)
- style.css (diseño futurista translúcido sin imágenes)
- script.js (lógica: login, reservas, roles, fotos guardadas en localStorage)

Cómo usar:
1) Extrae el ZIP.
2) Abre index.html con Live Server en VS Code (recomendado) o abre en navegador.
3) Usuarios demo:
   - admin / admin123
   - profesor / prof123
   - tecnico / tech123

Notas:
- Fotos se guardan en localStorage (base64) como demo.
- No hay imágenes externas; el diseño es puro CSS/SVG/JS.
